# 日本全国！方言励ましボタン

心が落ち込んでいる人に対して、地元の言葉で励ましてもらえるPWAアプリです。

## 機能

- 🗣️ **6つの方言モード**: 伊賀弁、沖縄弁、東北弁、関西弁、博多弁、土佐弁
- 📱 **PWA対応**: インストール可能、オフライン動作
- 🔊 **音声読み上げ**: Web Speech APIでメッセージを読み上げ
- 📤 **シェア機能**: Twitter/LINEなどでメッセージをシェア
- 🎯 **カテゴリー分け**: 疲れた時、落ち込んだ時、頑張りたい時など
- ✨ **エフェクト**: パーティクル、画面シェイク、サウンド、バイブレーション
- 🔥 **連続タップ機能**: コンボシステムで特別エフェクト

## デプロイ

このアプリはGitHub Pagesでデプロイされています。

### ローカルで実行

```bash
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開きます。

### GitHub Pagesでのデプロイ

1. リポジトリのSettings > Pagesに移動
2. Sourceを「GitHub Actions」に設定
3. mainブランチにプッシュすると自動的にデプロイされます

## インストール方法

### モバイル（iOS/Android）

1. ブラウザでアプリを開く
2. メニューから「ホーム画面に追加」を選択
3. アプリがインストールされます

### デスクトップ（Chrome/Edge）

1. ブラウザでアプリを開く
2. アドレスバーの「インストール」アイコンをクリック
3. アプリがインストールされます
